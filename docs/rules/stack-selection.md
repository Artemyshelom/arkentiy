# Выбор стека для задачи

| Задача | Инструмент | Почему |
|--------|-----------|--------|
| Простая выгрузка по расписанию | Google Apps Script | Бесплатно, встроенный таймер, прямой доступ к Sheets |
| Сложная логика, несколько API | Python/Docker | Гибкость, полный контроль, уже развёрнут |
| Webhook-обработка | Python/FastAPI | Нужен сервер — он есть |
| Быстрая связка без кода | n8n/Make | Визуально, но менее гибко |

Показывай Артемию плюсы/минусы — пусть выбирает.
