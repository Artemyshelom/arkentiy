## Бот-агенты Аркентия (на OpenClaw, morf-сервер)

**Платформа:** OpenClaw Node.js, работают в Telegram (приватные чаты + группы)  
**Auth:** каждый агент имеет отдельный Telegram bot token из `.../openclaw.json`  
**Data Access:** каждый имеет свой Stats API ключ (bearer token)

### 1. Мёрф — аналитический мозг

| Параметр | Значение |
|----------|----------|
| **ID в openclaw** | `main` (default) |
| **Роль** | Главный аналитик, создатель собственных идей, драйв проектов |
| **Scope** | Основные города Артемия (Абакан, Барнаул, Томск, Черногорск) |
| **Model** | `anthropic/claude-opus-4-5` |
| **Модули** | Все модули Аркентия + веб-поиск + дизайн Telegram UX |
| **Stats API key** | встроен в Kong (без bearer, auth по IP) |
| **Workspace** | `/opt/morf/workspace` |

**Чаты:**
- Основной отчёт: `-1003533462253` (без mention)
- `@mention` доступен везде (Артемию и др.)

**Специалист по:** big picture, переговоры, архитектура, стратегия — буквально "мозг" всей операции.

### 3. Станислав — консультант-наставник

| Параметр | Значение |
|----------|----------|
| **ID в openclaw** | `stanislav` |
| **Роль** | Консультант по системе, помогает новым управляющим разобраться в инструменте |
| **Scope** | Любой клиент (Артемий + будущие) |
| **Model** | `anthropic/claude-haiku-4-5` |
| **Modules** | Stats API + Exa веб-поиск (для качки примеров и трендов) |
| **Stats API key** | `stn_1038f90c5f16469444b9a602ce87fed13f33faad1c82` |
| **Workspace** | `/opt/morf/workspace-stanislav` |

**Чаты:**
- Активируются через `/api/consultant/activate` на arkentiy-сервере
- Таблица: `consultant_chats` (chat_id → tenant_id)

**Специалист по:** обучение, правильная интерпретация отчётов, нормативы, best practices в доставке еды.

**Особ:** 
- Может сам проверить цифру в Stats API чтобы подтвердить совет или привести пример
- Направляет к Борису только при запросе глубокой аналитики или регулярного мониторинга
- Знает KNOWLEDGE.md Аркентия (полная документация)

**Активация чата клиентом:**
```bash
curl -X POST "https://arkenty.ru/api/consultant/activate" \
  -H "Authorization: Bearer ADMIN_KEY" \
  -d '{"chat_id": -100..., "tenant_id": "artemiy"}'
```

## Specs — контракты перед реализацией
