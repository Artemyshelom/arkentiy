## Аудитория

## Структура страницы (сверху вниз)

### 1. Header (фиксированный)

```
[Логотип Аркентий]                    [Тарифы] [Как работает] [Попробовать бесплатно →]
```

- Логотип — текстовый, без картинки (на первое время)
- Навигация: якорные ссылки на секции + CTA-кнопка
- На мобайле: бургер-меню
- При скролле: header становится полупрозрачным с blur

### 2. Hero

```
Контроль доставки в реальном времени
Для ресторанов и сетей на iiko

Опоздания, отчёты, аудит, финансы — 
в Telegram-боте и на дашборде. За 5 минут.

[Попробовать 7 дней бесплатно]   [Рассчитать стоимость →]
```

- Заголовок: крупный, 1 строка (десктоп), 2 строки (мобайл)
- Подзаголовок: 2 строки, конкретика
- Два CTA: основной (триал) + вторичный (калькулятор → `/pricing`)
- Фон: нейтральный градиент или паттерн (не фото)

### 3. Проблема / Решение (3 колонки)

```
❌ Узнаёте об опоздании от клиента      →  ✅ Алерт в Telegram через 15 минут
❌ Отчёт за вчера — в конце недели       →  ✅ Автоматически в 9:00 каждый день
❌ Подозрительные отмены видны случайно  →  ✅ Аудит ежедневно — аномалии выделены
```

- 3 строки: боль → решение
- На мобайле: вертикальный список
- Цвета: красный для проблемы, зелёный для решения

### 4. Возможности / Модули (карточки)

```
[Карточка: Алерты]        [Карточка: Отчёты]        [Карточка: Поиск]
Опоздания +15/+30/+45     Утренний отчёт по всем    Заказ по номеру или
мин — сразу в Telegram     точкам с динамикой        телефону за секунды

[Карточка: Аудит]         [Карточка: Выгрузка]      [Карточка: Финансы ⭐]
Подозрительные отмены,     Заказы → Google Sheets    Сверка выписок банка
удаления, корректировки    автоматически             с кассой iiko

                          [Карточка: Конкуренты ⭐]
                          Мониторинг цен и меню
                          конкурентов в вашем городе
```

- 6 карточек базового пакета + 2 add-on (с пометкой "Доп. модуль")
- Каждая карточка: иконка + название + 1-2 строки описания
- Add-ons визуально отличаются (обводка или бейдж)
- На мобайле: горизонтальный скролл или 2 колонки

### 5. Как это работает (3 шага)

```
1. Подключите iiko          2. Добавьте бота           3. Получайте данные
   Укажите URL и логин         @arkentiy_bot в чат        Алерты, отчёты, аудит —
   — мы проверим за            вашей команды              автоматически каждый день
   30 секунд
```

- 3 шага горизонтально (десктоп) / вертикально (мобайл)
- Номера шагов крупно
- Минимум текста — по 2 строки

### 6. Социальное доказательство

```
9 точек подключено  ·  4 города  ·  Работает с февраля 2025

«Аркентий заменил нам ежедневную рутину по сбору отчётов.
 Менеджеры видят проблемы в реальном времени, не в конце дня.»
 — Ёбидоёби, сеть доставки суши
```

- Метрики в одну строку
- 1 цитата (можно будет добавить ещё)
- Без фотографий (пока)

### 7. Тарифы (превью)

```

[Начать бесплатно]

- Контрастный фон (тёмный блок)
- Одна кнопка

### 9. Footer

```

Telegram: @artemiy_shelomentsev     Email: support@arkentiy.ru

- Контактные данные

- Минимальный

## Стек и ограничения

**Фронтенд:** HTML + Tailwind CSS + минимальный JS (анимации при скролле, мобильное меню)

**Обоснование:** статическая страница, нет состояний, нет API-запросов. React избыточен.

**Адаптивность:** mobile-first (решение часто принимается с телефона)

**Браузеры:** Chrome, Safari (последние 2 версии)

**SEO:** мета-теги, Open Graph, structured data (Organization)

**Скорость:** целевой Lighthouse > 90 (нет тяжёлых ресурсов)
