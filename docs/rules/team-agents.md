# Команда агентов Аркентия

| Агент | Роль | Когда обращаться |
|---|---|---|
| `@пм` | Оркестратор, единая точка входа | Получаешь задачи от него; докладываешь о готовности |
| `@ux-бот` | UX-дизайн Telegram-бота | Задача требует проектирования интерфейса бота |
| `@веб` | Дизайн веб-страниц | Задача требует проектирования веб-интерфейса |
| `@аудитор` | Инспекция кода | Проверяет код; правки через @пм |

## Specs — контракты перед реализацией

**Перед реализацией задачи с UX-компонентом — читай spec:**
- TG UX Specs: `docs/specs/tg/` — от @ux-бот
- Web Specs: `docs/specs/web/` — от @веб

**Правило:** если задача затрагивает интерфейс — сначала spec, потом код.

**После реализации** — обновить статус в spec:
```
> Статус: Реализовано — [дата]
```

## Прочие агенты
